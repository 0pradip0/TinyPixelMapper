Open Stage Control

TPM Player OpenStage Control Config File for version 1.15.8

TPM Editor OpenStage Control Config File for version 1.15.8

On this SD in the Folder OSC you can find a copy of Open Stage Control

Open Stage Control OSC port = 9000 , TPM default address:port = 192.168.4.1:8000 if WifiAccess Point , DHCP on Ethernet

You can edit the Config file /conf/wifi.txt to set a IP or use Open Stage Control as a Gui

When Ethernet is connected Wifi is turned off. If we dont recive a DHCP ip over Ethernet it switches Wifi back on except if disable over the GUI
